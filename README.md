# zz-react-cli(zrc)

npm install zz-react-cli -g
zrc new yourProjrct --install
//mkdir yourProjrct&zrc init

* 集成自动ftp部署到测试服务器
* 集成rem
* 集成react-router异步路由
* 集成webpack chunk
* 集成eslint
* 集成数据mock
